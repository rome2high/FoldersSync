This Program will Synchronize two foldesrs in realtime

Usage1:
1. double click on .exe and follow command prompt screen
2. using command prompt, run the .exe and passing in 2 parameters one for each directory valid path


Usage2:
1. create a folder under C drive name "filesync": "C:\FileSync"
2. Add FileSync.config to the folder
3. Modify FileSync.config file to you need
4. Start application
